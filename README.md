# opencode-config

Global [opencode](https://opencode.ai) configuration.

## Structure

```
├── opencode.json    # Main config (model, plugins, MCPs)
└── command/         # Custom slash commands (Ralph Loop, PRD workflows)
```

## Setup

```bash
git clone git@github.com:jackwatters45/opencode-config.git ~/.config/opencode
```
